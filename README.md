## Installation

1. Clone the repo:
```console
$ git clone https://github.com/Shinyhero36/Spicetify-Taste-Analyzer
```

2. Add `taste` to the `CustomApps` folder
3. Run `spicetify config custom_apps taste`
4. Run `spicetify apply`
5. Enjoy
